Simple, intuitive API

Quick Start
Get up and running in seconds with just a few lines of code.
<gl-button motion="snappy">Click me</gl-button> <gl-input placeholder="Enter text"> <span slot="label">Label</span> </gl-input> <gl-card surface="glass" effect="tilt"> <div slot="header">Card Title</div> <div>Card content</div> </gl-card>
Customization
Override colors, spacing, and more with CSS custom properties.
:root { --gl-primary: #693B93; --gl-radius: 16px; --gl-space-4: 20px; }
Framework Integration
Works seamlessly with React, Vue, Angular, and more.
